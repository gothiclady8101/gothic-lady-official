# GOTHIC LADY OFFICIAL WEBSITE

AIシンガー「ゴシックレディ」の公式ポータルサイト。
「漆黒と深紅」をテーマにしたサイバーゴシックな世界観を、モダンなWeb技術で表現しています。

## 特徴

- **Cyber Gothic Design**: EGOISTのスタイルをモデルにした、高コントラストでデジタルな質感のUI。
- **Real-time Data Sync**: Sunoのプロフィールから最新の楽曲・ジャケット画像を表示。
- **Immersive Effects**:
  - ページ読み込み時の「ヴィジョン（幻影）」出現アニメーション。
  - マウス追従型のカスタムカーソルと文字グリッチ演出。
  - 背景で舞い落ちる薔薇の花びら（パーティクルエフェクト）。
- **Mobile Optimized**: PC/タブレット/スマートフォンすべてのデバイスに最適化済み。

## 開発環境

- HTML5 / CSS3 (Vanilla JS)
- Google Fonts: `Shippori Mincho`, `Rajdhani`

## 公開方法 (GitHub Pages)

1. GitHubリポジトリにこのファイルをアップロードします。
2. Settings > Pages > Build and deployment に移動します。
3. Branchを `main` に設定し、`docs` フォルダまたは `root` を選択して保存します。
4. 数分後、提供されたURLでサイトが公開されます。
